# is-hackathon 
Petit site vitrine pour un service de protection : des pages HTML unique articulée autour d’un hero visuel mettant en avant la promesse de sécurité, de sections descriptives (services 24/7, intervention rapide, conseils personnalisés) et d’un formulaire de contact validé en JavaScript pour capter les prospects, le tout stylé avec une feuille CSS responsive (palette sombre/bleu, grilles flex) et illustré par une image principale, afin d’offrir une présentation claire, rassurante et prête à être enrichie (témoignages, stats, animations ou intégration d’une API d’envoi).
